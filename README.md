## Features

- **Bilingual Support** — Seamlessly switch between English and Portuguese
- **Responsive Design** — Optimized for desktop, tablet, and mobile devices
- **Scroll Animations** — Smooth reveal effects as you explore the page
- **Dark Terminal Theme** — Styled with a tech-inspired aesthetic
- **Project Showcase** — Display of completed and unreleased projects
- **Contact Integration** — Easy email and social media links

## Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Styling:** Custom CSS with CSS variables for easy theming
- **Fonts:** JetBrains Mono (monospace), Font Awesome (icons)
- **Animations:** CSS transitions and IntersectionObserver API
- **Localization:** Language switching system with localStorage persistence
